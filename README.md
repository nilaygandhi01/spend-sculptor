# SpendSculptor

SpendSculptor is a procurement / spend analytics application. This repository holds the **SpendSculptor** codebase and related assets (Python backend, web frontend, configuration, and data files as needed).

## Repository layout

Add your application files here, for example:

- Python modules and scripts (`.py`)
- Frontend assets (`*.html`, `*.js`, `*.css`)
- Dependency manifests (`requirements.txt`, `package.json`, lockfiles)
- Configuration (e.g. `pyproject.toml`, `.env.example`, Docker files)
- Sample or generated data only when appropriate for the app (never commit secrets)

## Setup

1. Create a virtual environment and install dependencies (e.g. `pip install -r requirements.txt` when that file exists).
2. Copy `.env.example` to `.env` and fill in values (do not commit `.env`).
3. Run the app per instructions you add in this README once the source tree is in place.
